TOKANTINS BIXOPEDIA - VERSAO PARA PUBLICAR

Para o app aparecer na tela inicial do celular com icone:

1. Publique todos os arquivos desta pasta em um servico com HTTPS.
   Exemplos: Netlify, GitHub Pages, Vercel ou uma hospedagem propria.

2. Abra o link publicado no navegador do celular.

3. No Android/Chrome:
   toque no menu do navegador e escolha "Adicionar a tela inicial" ou "Instalar app".

4. No iPhone/Safari:
   toque em compartilhar e escolha "Adicionar a Tela de Inicio".

Nao envie apenas o index.html pelo WhatsApp. O app precisa desta pasta inteira,
incluindo styles.css, app.js, data.js, manifest.webmanifest, sw.js e assets.

O icone usado para instalacao e a mini logo TO em assets/icons.
